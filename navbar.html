Navigation Bars

How do you make navigation bars? It's pretty simple.

At a very basic level, navbars are nothing more than some words that are also a links. If you know the most basic level of using link tags, you can make a navigation system. I'll show you one solid clean way.

Generally there are two types of navigation on websites: Vertical and Horizontal Navigation. I'll show you a simple way to do both, starting with vertical:

HTML
<!DOCTYPE html> <html> <head> <meta charset=utf-8 /> <title>My Website</title> <link type="text/css" rel="stylesheet" href="css/reset.css"/> <link type="text/css" rel="stylesheet" href="css/mainstylesheet.css"/> </head> <body> <ul> <li class="navitem">Fish</li> <li class="navitem">Minerals</li> <li class="navitem">Balloon Hats</li> <li class="navitem">Trumpets</li> </ul> </body> </html>
CSS
.navitem { background-color: cyan; height: 25px; width: 110px; font-size: 20px; padding-left: 10px; }

So what's going on? At the base we have a list, <ul> is an unordered list element. Inside it are the four navigation words, each one of them is a <li> or list item. So it's just a list with four items. I've given the list items a class so we can use CSS to change how they look (class navitem).

So the HTML has already created a vertical list for me, it's what it does by default. Then we just changed the background color of each item to cyan and gave them a width and a height (navigation buttons should be easy to click). Next we made the font larger, and finally we added a bit of padding to the left side of each item. The padding adds 10px to the left, effectively pushing the text over a bit so it's not right up against the left edge of the page. Obviously you can edit these parameters however you like. Want giant pink 120px by 120px square navigation buttons? Go for it.

Ok, So now we have decent buttons, but they don't do anything. Let's make them into links:

HTML
<!DOCTYPE html> <html> <head> <meta charset=utf-8 /> <title>My Website</title> <link type="text/css" rel="stylesheet" href="css/reset.css"/> <link type="text/css" rel="stylesheet" href="css/mainstylesheet.css"/> </head> <body> <ul> <a href="http://www.google.com"><li class="navitem">Fish</li></a> <a href="http://www.facebook.com"><li class="navitem">Minerals</li></a> <a href="http://www.myspace.com"><li class="navitem">Balloon Hats</li></a> <a href="http://www.disneyland.com"><li class="navitem">Trumpets</li></a> </ul> </body> </html>

All that we've done here is to wrap each list item in a <a> (link) tag.

The href="" attribute is where we put the address of wherever we want the link to go (between the quotes). In this case I just put Google and Facebook etc. but you would add the adresses of your pages, or whatever you like.

So now your buttons should work as links, but they also probably look like this:

All that happened here is that there is a default color (blue) and underline on links. Since we probably don't want this in our navbar, all we have to do is remove it with this CSS:

CSS
.navitem { background-color: cyan; height: 25px; width: 110px; font-size: 20px; padding-left: 10px; color: black; text-decoration: none; }

So we set the blue text color back to black, and removed the underline text decoration. Back to normal.

Great, so now we've got a completely functional vertical navbar! If you wanted to add more items you could just add more <li>'s wrapped in <a>'s just like the four previous ones.

So how do we make a horizontal navbar? Same way we make most other things that stack vertically stack horizontally. Just add one CSS line:

CSS
.navitem { background-color: cyan; height: 25px; width: 110px; font-size: 20px; padding-left: 10px; color: black; text-decoration: none; display: inline-block; }

Wow, that was sooooooo easy. If you wanted you could take this horizontal navbar and paste it right into the blue navbox spot on the standard page layout (copy the whole ul and li's inside it, and dont forget the "navitem" CSS rules), and it would work perfectly:

HTML
<!DOCTYPE html> <html> <head> <meta charset=utf-8 /> <title>My Website</title> <link type="text/css" rel="stylesheet" href="css/reset.css"/> <link type="text/css" rel="stylesheet" href="css/mainstylesheet.css"/> </head> <body> <div class="midwrap"> <div class="header"></div> <div class="navbox"> <ul> <a href="http://www.google.com"><li class="navitem">Fish</li></a> <a href="http://www.facebook.com"><li class="navitem">Minerals</li></a> <a href="http://www.myspace.com"><li class="navitem">Balloon Hats</li></a> <a href="http://www.disneyland.com"><li class="navitem">Trumpets</li></a> </ul> </div> </div> </body> </html>
CSS
html { height: 100%; }
body { height: 100%; }
.navitem { background-color: cyan; height: 25px; font-size: 20px; padding-left: 20px; width: 110px; display: inline-block; }
.midwrap { min-height: 100%; width: 1000px; background-color: red; margin: auto; }
.header { width: 1000px; height: 120px; background-color: green; }
.navbox { width: 1000px; height: 40px; background-color: blue; }

Beautiful, you might want to adjust the size of the buttons to fit better, just a quick CSS edit of the .navitem will allow any kind of customizations you desire. The navbar is its own little self contained module. All you have to do is make a space for it and paste it in wherever.
