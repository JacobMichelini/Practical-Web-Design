Practical Web-Design

Aesthetics

Alright, so now that you have a basic understanding of how to structure a functional page, it's time to make it look like a real page.

Basically this is all about just remembering a bunch of CSS properties that can be used to make a nice looking effect, so I'll just save you some time and list a bunch of them and how they can look. If you need more specific information on properties and what values they will accept, search the property on W3schools.com.

Box Styling

Here are the properties that can effect a block element like a div.

Background Color

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; background-color: red; }
BROWSER

Simple, Changes the background color of the box. Can enter the value in basic named colors like red/blue, hex values like #E32020, or RGB values like rgb(255,0,0).

Background Image

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; background-image: url(wood.jpg); }
BROWSER

Sets the background to a picture, in this case wood.jpg located in the same folder as the CSS file. This effect is used to create very slick looking webpages when each div has a custom created image skinned onto it. It's all down to your photoshop and art skill at this point. You can use images with transparency in them for cool effects such as making the box appear to not be rectangular.

Borders

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; border: 2px solid black; }
BROWSER

Adds a border. The first value in the CSS is the border's width, the second is the style (there are a few options such as solid, dotted etc), and the third is the color of the border. Borders can also be applied only to certain side of the element with border-left, border-right, etc.

Rounded Corners

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; border-radius: 10px }
BROWSER

Rounds the corners of a box. Values can be in pixels or in percentages. A value of 50% will turn a square into a circle, although keep in mind that it will only look like a circle, it will still be a box for document flow purposes.

Box Shadows

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; Box-shadow: 2px 2px 5px 0px black; }
BROWSER

Adds a shadow. The first value moves the shadow horizontally. The second Value moves the shadow vertically. The third value is blur amount (higher is more blur). The fourth value expands or contracts the shadow (you can leave this out if you don't need it). The fifth is the shadow's color. You can add the property "inset" after the color to change the shadow from outside the box to inside it. Shadows are a very flexible tool for a huge amount of effects. With some creativity you can make them into a glowing effect with a light color, or make a blurry effect with a visible shadow for a transparent box, or all sorts of things. If you set the blur to zero you can even use them to create fake boxes.

Tilting

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; transform: rotate(15deg); -ms-transform: rotate(15deg); -webkit-transform: rotate(15deg); }
BROWSER

Rotates any element. May not work on older browsers. You must use all three lines for the effect to be compatible with IE, Chrome, and Firefox etc. Keep in mind that the element only looks rotated, it is still in its origninal position for box model purposes.

Opacity

CSS
.box { height: 100px; width: 100px; margin-left:8px; margin-top:8px; opacity: .6; }
BROWSER

This will make your element tranparent. Obnoxiously it will also make all elements inside the element also transparent, so making solid black text inside a partially transparent div will not work. To achieve that kind of effect you can try using a background color in RGB with an alpha channel, or using a partially transparent background-image, among other tricks.
